# Fruit-Capture-Asteroid-Dodge

This is a simple terminal based game developed in python.

Play the Games using arrow keys or number keys - 2 (UP), 4(Left), 6(Right), 8(Down)
It has two modes:
 1. Fruit Capture - 'U' catches the dropping items 'O' 

&nbsp;2. Asteroid Dodge - 'A' dodges the coming obstacles 'O'



To play, just copy the repository then use the command : py curserGme.py \[option]
use option -g \[1 or 2] to switch between games

(or 'python3' in place of 'py')
