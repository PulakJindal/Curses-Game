This is a simple terminal Game where the player 'U' captures the falling number 'O'



Just Copy the repository, enter to the folder location through your terminal (cmd or Powershell or bash) and run the command  - py cursesGame.py
(You can change the 'py' to 'python' or 'python3' according to the version of python you have in your system)
